# **App Name**: TabTune Animator

## Core Features:

- Code Import: Import Motion Canvas code snippets to define animations.
- Real-time Preview: Render a preview of the animation based on the provided code.
- Tab Input: Enter tab notation to automatically generate guitar tab animations using generative AI as a tool to select optimal framings.
- Video Export: Export the animation as a video file with adjustable resolution and frame rate.
- Customization: Allow users to adjust the colors and shapes to match their brand

## Style Guidelines:

- Primary color: Dark, muted blue (#34495E) to provide a professional and focused atmosphere.
- Background color: Even darker desaturated blue (#222F3D), near black, for high contrast against UI elements.
- Accent color: A vibrant teal (#00BCD4) to highlight interactive elements and calls to action.
- Body and headline font: 'Space Grotesk' sans-serif font for clear code representation and UI elements.
- Code font: 'Source Code Pro' monospaced font for the code editor and any displayed code snippets.
- A clear layout separates the input/editor, preview, and output sections for efficient workflow.
- Use subtle transitions and animations to provide feedback during user interactions, without distracting from the code or animation preview.
